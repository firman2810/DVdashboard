# FinScope: Credit Card Fraud Detection & Customer Spending Analysis Dashboard

## Overview

FinScope is an interactive Business Intelligence dashboard developed for the **TEB3133/TFB3133 Data Visualization Project**. The dashboard focuses on the **Finance** domain by combining **Credit Card Fraud Detection** and **Customer Spending Analysis** into a unified visualization platform.

The dashboard enables users to identify fraud patterns, analyse customer spending behaviour, and generate actionable insights through interactive visualizations and advanced analytics.

---

## Objectives

The project aims to:

- Develop an interactive dashboard for visualizing credit card fraud indicators and customer spending behaviour.
- Design a user-centred dashboard that supports different categories of users with varying analytical needs.
- Implement interactive visualization features that enable efficient exploration of multivariate financial transaction data.

---

## Dataset

The dashboard uses a **Credit Card Transaction Dataset** containing:

- **100,000 transaction records**
- **21 attributes**

The dataset includes:

- Transaction Date & Time
- Transaction Amount
- Merchant Category
- Customer Age
- Customer Gender
- Customer Location
- Merchant Information
- Fraud Indicator (`is_fraud`)
